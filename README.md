# 🏁 Simulador de Corrida
> Projeto desenvolvido para praticar e consolidar conhecimentos de Programação Orientada a Objetos (POO) em Java.
> Simula uma corrida entre dois veículos com a utilização do recurso "math.random" para tornar o resultado imprevisível.

## 💻 Tecnologias utilizadas:
* Java
* Eclipse IDE
* Git e GitHub

## ⚙ Funcionalidades: 
* 🏆 Demonstrar pista
* 🏎️ Simular corrida
